About the YouthMappers Validation Hub

The HubA monitoring & synchronization workspace

This validation hub is a custom, high-visibility monitoring and synchronization workspace designed and maintained by Brazil Singh (YouthMappers Regional Ambassador · Director of the OpenStreetMap Foundation).

It operates as a telemetry aggregator and validation coordination workspace, directly monitoring YouthMappers mapping campaigns on the global Humanitarian OpenStreetMap Team (HOT) Tasking Manager, with TeachOSM instance support on the roadmap. By polling live REST API endpoints, it tracks regional campaigns dedicated to disaster relief, infrastructure building, and localized sustainable development datasets — helping prevent duplication of effort and coordinate volunteer validation channels.

The NetworkAbout YouthMappers

YouthMappers is a global network of university student chapters organized to map for development and humanitarian resilience. Formed in 2015, the network mobilizes student leadership by creating and using free, open-source geographic data, supporting local and international development programs in public health, municipal planning, and climate adaptation.

Supported by the USAID GeoCenter, West Virginia University, Texas Tech, and George Washington University, the consortium has grown to hundreds of campus-led chapters worldwide.

ArchitectureLive REST API poller

Queries the production Tasking Manager API for campaigns matching organisationName=YouthMappers dynamically, paginating through every published project and re-syncing on a five-minute cycle.

ArchitectureRead-only, privacy-first pipeline

All data travels over HTTPS (TLS-encrypted) from HOT's public API with read-only scope. Rendering happens entirely in your browser — no backend, no accounts, no tracking, and nothing written back, so the source data can never be polluted.

ArchitectureBento workspace layout

Consolidates live project grids, validator-need indicators, filter chips, and contributor statistics into one beautiful landscape workspace that scales from mobile to widescreen wall displays.
