# Security Policy

Thank you for helping keep the **YouthMappers Validation Hub** secure. We take the security and privacy of our users seriously and appreciate responsible disclosure of security vulnerabilities.

## Supported Versions

The following versions are currently supported with security updates.

| Version | Supported |
| ------- | :-------: |
| 1.4.0 | ✅ |
| 1.0.1 | ❌ |

> Please update to the latest release before reporting issues that may have already been fixed.

---

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not create a public GitHub Issue**.

Instead, report it privately through our GitHub Discussions security channel:

**https://github.com/brazilsingh/youthmappers-validation-hub/discussions/categories/show-and-tell**

When reporting, please include as much information as possible:

- Description of the vulnerability
- Steps to reproduce
- Expected behaviour
- Actual behaviour
- Browser and operating system
- Screenshots or proof-of-concept (if applicable)
- Any suggested mitigation

---

## What to Expect

After receiving your report, we aim to:

- Acknowledge receipt within **72 hours**
- Review and validate the report
- Keep you informed of the investigation progress
- Release a security fix as quickly as practical if the issue is confirmed
- Publicly acknowledge your contribution (if you would like to be credited)

If the reported issue is determined **not to be a security vulnerability**, we will explain our reasoning and, where appropriate, suggest opening a normal GitHub Issue instead.

---

## Supported Security Features (v1.4.0)

Current security-related improvements include:

- Dark/Light mode preference stored locally in the browser (no user tracking)
- No server-side user authentication or personal data storage
- HTTPS deployment via GitHub Pages
- Dependency updates through GitHub
- Open-source codebase for community review
- Security updates provided only for the latest stable release (1.4.x)

---

## Responsible Disclosure

Please allow us reasonable time to investigate and resolve security issues before publicly disclosing them.

We appreciate responsible disclosure and thank everyone who helps improve the security of the YouthMappers Validation Hub.
